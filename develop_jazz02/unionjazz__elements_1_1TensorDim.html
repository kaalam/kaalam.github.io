Detailed Description

The dimension of a tensor.

The structure is declared as a union to make filter operation more elegant. A filter is a record that always has rank 1 and a size, but has an extra parameter, its length. Since dim[1] is not used (as rank is 1), it is a good place to store the length but remembering that dim[1] is the length is ugly. Therefore, the more elegant filter.length is an alias for dim[1].
