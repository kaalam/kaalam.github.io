The Basic Opcode Pack (BOP)

Snippets: Snippets and code items. (A code item is a minimal snippet.)

Primitives: stateless function, stateful function, mutable function, side-effect function.

Packs: A service to dynamically link Pack collections.

Pack: A collection of primitives.

Formal Fields: Organizing primitives and snippets by Kind.

Core: The execution unit running Snippet.

The fundamental collection of primitives provided in Jazz (any other pack is an uplifted pack).

A core is not a service, it allocates its state in the Volatile container.

Fields are morphisms defined as primitives and snippets in a unique way for each pair of Kinds.

Packs are collections of primitives, either the Basic Opcode Pack (BOP), or a linked dynamic library referred to as an uplifted pack.

This Service manages uplifted Packs.

These functions are known as primitives (or op-codes) because they are implemented in C++ inside a Kernel.

Functionally, snippets can do the same as primitives, but they are written in Bebop rather than C++.
