Detailed Description

Simplest functionality to operate as an http client to GET, PUT, DELETE JazzBlock descendants to/from other Jazz nodes or any http servers.

< This module uses curl in combination with JazzBlock and Jazz allocators (one shot or volatile) to simplify the communication from remote sources or across Jazz nodes.
