Detailed Description

Arithmetic, logic and type conversion stdcore applicable to JazzDataBlock structures.

< This module defines pure functions that accept JazzDataBlock structures as arguments and return JazzDataBlock structures or throw exceptions. These functions are called "stdcore" for a reason, they are the simplest blocks approximately matching Bebop bytecode instructions.
