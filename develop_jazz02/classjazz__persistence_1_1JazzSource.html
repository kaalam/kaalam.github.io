Detailed Description

A much simpler interface to create/read/update/delete JazzBlocks in source using and underlying JazzPersistence that can be exported to R, Python and the REST API directly.

THREAD SAFETY: All public methods in JazzBlockKeepr descendants must be thread safe. In the core objects, thread-safe failure in public methods is treated as a top priority bug that is intended to be spotted in burn-in tests. Private methods can be unsafe, but the public methods calling them must be aware of their limitations and use thread-locking when necessary. (Copy this message in all descendants.)
