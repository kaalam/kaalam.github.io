◆ JazzBlockMap

typedef std::map<JazzBlockId64, const JazzBlockKeeprItem *> jazz_containers::JazzBlockMap

A map to search a JazzBlockKeeprItem by the block_id64 of its corresponding JazzBlock. This is the essential element that converts a queue into a cache. A cache is a queue where elements can be searched by id and lower priority items can be removed to allocate space for LRU (or higher priority items where "priority" is a real value assigned arbitrarily).

◆ close_jazz_block()

void jazz_containers::close_jazz_block ( pJazzBlock  p_block)
inline

Set the creation time and the hash64 of a JazzBlock

Despite its name, this function does not actually "close" anything. JazzBlock manipulation is based on "good will", after calling close_jazz_block() the owner should not change the content or should close_jazz_block() again after doing so. 

close_jazz_block() can be called any number of times on the same block.

\param p_block The block to be "closed".
