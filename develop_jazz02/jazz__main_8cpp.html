◆ main()

int main ( int  argc,
char *  argv[] 
)

Entry point.

Parameters
argc(Linux) number of arguments, counting the name of the executable.
argv(Linux) each argument, argv[0] being the name of the executable.
Returns
EXIT_FAILURE or EXIT_SUCCESS

When the command is "start": if running already, message + EXIT_FAILURE if configuration declared, but does no exist, message + EXIT_FAILURE else try to start the server calling main_server_start()

When the command is "stop": if not running already, message + EXIT_FAILURE else send a SIGTERM to the server and wait to check if it closes if the server closes before 20 seconds, message + EXIT_SUCCESS else message + EXIT_FAILURE

When the command is "status": if not running already, message + EXIT_FAILURE else message + EXIT_SUCCESS

When the command is anything else, too many or too few: show help + EXIT_FAILURE
