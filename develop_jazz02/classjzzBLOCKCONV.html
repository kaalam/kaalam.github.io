bool jzzBLOCKCONV::cast_lmdb_block ( int  source,
persistedKey  key,
int  type 
)

Implement the API function PUT //source.key.header/type to set the block type to a compatible type.

Parameters
sourceThe index of an open source. (As returned by get_source_idx().)
keyA key identifying the block in that source.
typeThe new type to which the block will be converted.
Returns
true if successful, false and log(LOG_MISS, "further details") if not.

This function is a part of the APIs exec_block_put_function() separated to avoid that function becomes unmaintainable. What it basically does is calling reinterpret_cast_block() and hash_block() and rewriting the block handling all the error/pointer logic.

bool jzzBLOCKCONV::translate_block_FROM_R ( pJazzBlock  psrc,
pJazzBlock &  pdest 
)

Create a block_C_BOOL, block_C_R_INTEGER, block_C_R_REAL or block_C_OFFS_CHARS from a block_C_R_RAW of type BLOCKTYPE_RAW_R_RAW binary compatible with a serialized R object. 

\param psrc  An R object serialized as a block_C_R_RAW of type BLOCKTYPE_RAW_R_RAW
\param pdest Address of a pJazzBlock allocated by this function to store the block.

\return      true if successful, false and log(LOG_MISS, "further details") if not.

This function does nothing with the psrc object except copying it.
The returned pdest is owned by the caller and must be JAZZFREE()ed when no longer necessary.
