◆ load_or_create_space()

StatusCode jazz_models::SemSpace::load_or_create_space ( )
private

Load or create the space.

This is called once when start() has successfully completed. It will load from persistence (and not write into persistence) if def.load_on_start. Otherwise, if the table metadata exists, it will fail. To override a table, you must use the SemSpace-ETL interface.- Otherwise, it will create the table using the definition in def.

Returns
SERVICE_NO_ERROR if successful, an error code otherwise.
