Detailed Description

ExtraLocator: A structure that replaces the entity/key in a Locator by a long URL or file name and some http quirks.

Valid characters**: Most non-zero characters should be usable or will be rejected by the endpoint, not the parser. See the doc on Channels::as_locator for details.
