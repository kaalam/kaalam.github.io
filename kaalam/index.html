Massa libero

Yesterday’s breakthrough applications were driven by big data – tomorrow’s breakthrough applications will be driven by leveraging data with AI in real-time to billions of users. A light-weight server including persistence, data collection, crunching with deep neural networks, APIs, web and mobile all served by the same process is the only requirement to make it happen. Building on the shoulders of giants, free software and C++ makes such integration possible. We present Jazz, a lightweight all-in-one analytical and AI server created from a code base donated by BBVA Data & Analytics.
