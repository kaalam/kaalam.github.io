Detailed Description

Ancestor class of all Jazz services requiring start/stop/reload. This is an empty container of virtual methods that should be implemented in the descendants.

bool jazzService::reload ( )
virtual

Start a jazzService descendant.

This is the first method of a jazzService to be called.

If start() returns true, it MAY get reload() calls and WILL get just one stop() call. If start() returns false, it will never be called again.

Virtual method to be overridden in all objects implementing the interface. (I.e., descending from jazzService)
