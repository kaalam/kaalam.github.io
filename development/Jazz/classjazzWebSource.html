jazzWebSource::jazzWebSource ( )

Root class providing all the necessary web resource functionality.

This class keeps a dictionary of known urls as keys, the values include: block name Content-Length, Contenct_Language, Content_Encoding and Last-Modified. The source name is always "www". These dictionaries are stored in persistence.

The urls and all their attributes are stored in URLattrib structures. An abstract entity known as a websource organizes the links in the persistence in a way that a whole websource can be queried, created, deleted. websources are identified by name (websourceName) or index in a in internal vector queried via find_websource(). websources do no exist for the API, they are just a clean way to manage sets of urls that are created/removed together.
